# QuizMaster – Self Learning MCQ System

QuizMaster is a full-stack web-based MCQ learning platform developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript.

---

# Features

## User Features
- User Registration & Login
- Secure Authentication
- Start Quiz by Category
- Difficulty Level Selection
- Quiz Result Tracking
- Quiz History
- Performance Analytics
- Responsive Dashboard

## Admin Features
- Admin Login
- Manage Users
- Manage Categories
- Add/Edit/Delete Questions
- View Quiz Attempts
- Analytics Dashboard

---

# Tech Stack

- HTML5
- CSS3
- Bootstrap
- JavaScript
- PHP
- MySQL

# Installation

## 1. Clone Repository

```bash
git clone https://github.com/mayukt12/quizmaster.git
```

## 2. Move Project

Move project folder into:

```text
htdocs/
```

## 3. Start XAMPP

Start:
- Apache
- MySQL

## 4. Import Database

- Open phpMyAdmin
- Create database:
```text
quizmaster
```

- Import:
```text
quizmaster.sql
```

## 5. Run Project

Open browser:

```text
http://localhost/quizmaster
```
