# QuizMaster – Self Learning MCQ System

QuizMaster is a full-stack web-based MCQ learning platform developed using PHP, MySQL, Bootstrap, HTML, CSS, and JavaScript.

## Features
- User Registration & Login
- Admin Dashboard
- Quiz System
- Performance Analytics
- Quiz History
- Result Tracking
- Category Management
- Question Management
- Responsive Design

## Tech Stack
- HTML
- CSS
- Bootstrap
- JavaScript
- PHP
- MySQL
